Library Management System

A web-based Library Management System built using Java Servlet, JSP and MySQL.

Tech Stack
- Java
- Java Servlet and JSP
- JDBC
- MySQL
- HTML, CSS and Bootstrap

Features
- Add, View, Update and Delete Books
- Member Management
- Issue and Return Books
- Database connected with MySQL

How to Run
1. Clone the repository
2. Import project in Eclipse
3. Setup MySQL database
4. Configure DB connection in DBConnection.java
5. Deploy on Apache Tomcat
6. Open browser and go to http://localhost:8080/LibraryManagement
